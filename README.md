# Claude Marketplace

Plugin marketplace registry for [Saari](https://github.com/saari-ai). Registers and hosts Claude Code plugins from the Saari organization.

## What this is

A marketplace definition that lists available Claude Code plugins. Claude Code users can install plugins from this marketplace to extend their workflow with Saari tools.

## Available plugins

| Plugin | Description |
|--------|-------------|
| [kilpi-skills](https://github.com/saari-ai/kilpi-skills) | Interactive skills for applying the Kilpi security methodology |

## Structure

```
.claude-plugin/
  marketplace.json   # Marketplace metadata and plugin registry
```
